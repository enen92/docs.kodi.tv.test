Detailed Description

Base class for joystick driver primitives

A driver primitive can be:

  1. a button
  2. a hat direction
  3. a semiaxis (either the positive or negative half of an axis)
  4. a motor
  5. a keyboard key
  6. a mouse button
  7. a relative pointer direction

The type determines the fields in use:

Button:

  • driver index

Hat direction:

  • driver index
  • hat direction

Semiaxis:

  • driver index
  • center
  • semiaxis direction
  • range

Motor:

  • driver index

Key:

  • key code

Mouse button:

  • driver index

Relative pointer direction:

  • relative pointer direction
