Detailed Description

Inputstream color primaries flags
Chromaticity coordinates of the source primaries. These values match the ones defined by ISO/IEC 23001-8_2013 ยง 7.1.

Used on kodi::addon::InputstreamInfo::SetColorPrimaries and kodi::addon::InputstreamInfo::GetColorPrimaries.
