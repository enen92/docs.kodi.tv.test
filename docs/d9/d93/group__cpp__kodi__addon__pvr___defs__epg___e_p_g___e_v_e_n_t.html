Detailed Description

EPG entry content event types.
These ID's come from the DVB-SI EIT table "content descriptor" Also known under the name "E-book genre assignments".

See ETSI EN 300 468 V1.14.1 (2014-05) about.

Values used by this functions:

  • kodi::addon::PVREPGTag::SetGenreType()
  • kodi::addon::PVREPGTag::SetGenreSubType()
  • kodi::addon::PVRRecording::SetGenreType()
  • kodi::addon::PVRRecording::SetGenreSubType()

Following types are listed here:

emum Type Description
EPG_EVENT_CONTENTMASK EPG entry main content to use.
EPG_EVENT_CONTENTSUBMASK_MOVIEDRAMA EPG entry sub content to EPG_EVENT_CONTENTMASK_MOVIEDRAMA event types for sub type of "Movie/Drama".
EPG_EVENT_CONTENTSUBMASK_NEWSCURRENTAFFAIRS EPG entry sub content to EPG_EVENT_CONTENTMASK_NEWSCURRENTAFFAIRS event types for sub type of "News/Current affairs".
EPG_EVENT_CONTENTSUBMASK_SHOW EPG entry sub content to EPG_EVENT_CONTENTMASK_SHOW event types for sub type of "Show/Game show".
EPG_EVENT_CONTENTSUBMASK_SPORTS EPG entry sub content to EPG_EVENT_CONTENTMASK_SPORTS event types for sub type of "Sports".
EPG_EVENT_CONTENTSUBMASK_CHILDRENYOUTH EPG entry sub content to EPG_EVENT_CONTENTMASK_CHILDRENYOUTH event types for sub type of "Children's/Youth programmes".
EPG_EVENT_CONTENTSUBMASK_MUSICBALLETDANCE EPG entry sub content to EPG_EVENT_CONTENTMASK_MUSICBALLETDANCE event types for sub type of "Music/Ballet/Dance".
EPG_EVENT_CONTENTSUBMASK_ARTSCULTURE EPG entry sub content to EPG_EVENT_CONTENTMASK_ARTSCULTURE event types for sub type of "Arts/Culture (without music)".
EPG_EVENT_CONTENTSUBMASK_SOCIALPOLITICALECONOMICS EPG entry sub content to EPG_EVENT_CONTENTMASK_SOCIALPOLITICALECONOMICS event types for sub type of "Social/Political issues/Economics".
EPG_EVENT_CONTENTSUBMASK_EDUCATIONALSCIENCE EPG entry sub content to EPG_EVENT_CONTENTMASK_EDUCATIONALSCIENCE event types for sub type of "Education/Science/Factual topics".
EPG_EVENT_CONTENTSUBMASK_LEISUREHOBBIES EPG entry sub content to EPG_EVENT_CONTENTMASK_LEISUREHOBBIES event types for sub type of "Leisure hobbies".

EPG_GENRE_USE_STRING 

Used to override standard genre types with a own name about.

Set to this value EPG_GENRE_USE_STRING on following places:

  • kodi::addon::PVREPGTag::SetGenreType()
  • kodi::addon::PVREPGTag::SetGenreSubType()
  • kodi::addon::PVRRecording::SetGenreType()
  • kodi::addon::PVRRecording::SetGenreSubType()
Warning
Value here is not a ETSI EN 300 468 V1.14.1 (2014-05) conform.
Note
This is a own Kodi definition to set that genre is given by own string. Used on kodi::addon::PVREPGTag::SetGenreDescription() and kodi::addon::PVRRecording::SetGenreDescription()
