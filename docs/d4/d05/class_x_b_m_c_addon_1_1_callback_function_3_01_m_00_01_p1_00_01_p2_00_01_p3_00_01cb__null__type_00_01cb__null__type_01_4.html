Detailed Description

template<class M, typename P1, typename P2, typename P3>
class XBMCAddon::CallbackFunction< M, P1, P2, P3, cb_null_type, cb_null_type >

This is the template to carry a callback to a member function that returns 'void' (has no return) and takes three parameters.
